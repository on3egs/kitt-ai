# MODULE ECONOMY & ENERGY — Analyse Macroéconomique

## Indicateurs clés à suivre
- Taux directeurs : FED (USA), BCE (Europe), BOJ (Japon), BOE (UK)
- Inflation CPI / PPI / PCE
- Chômage U3 (officiel) / U6 (réel USA)
- PIB : croissance trimestrielle, dette/PIB
- Yield curve : inversion = signal récession (T2Y vs T10Y)
- DXY : indice dollar — corrélation inverse matières premières
- VIX : volatilité S&P500 — indicateur stress marché

## Énergie
- Pétrole : Brent (Europe), WTI (USA) — spread = tension logistique
- Gaz naturel : TTF (Europe) — très sensible géopolitique (Russie)
- Transition énergétique : LCOE solaire/éolien < charbon/gaz (2024)
- Stockage : batterie prix 100$/kWh seuil compétitivité VE atteint
- Uranium : renaissance nucléaire (France, USA, Japon) — prix spot × 3 depuis 2020

## Risques structurels mondiaux
- Dette souveraine : USA 120% PIB, Japon 260% PIB — soutenabilité à taux élevés
- Dédollarisation : marginal mais trend BRICS (Yuan, or, monnaies locales)
- Démographie : vieillissement EU/Japon/Chine → pression retraites, déflation
- Deglobalisation : reshoring supply chains → inflation structurelle
- Crypto : Bitcoin store of value institutionnel (ETF spot USA 2024)

## Cycles économiques
- Expansion → Pic → Contraction → Creux → Expansion
- Cycle Kondratiev (50–60 ans) : innovation technologique
- Cycle Juglar (7–11 ans) : investissement
- Cycle Kitchin (3–5 ans) : stocks/inventaires

## Géopolitique économique
- Sanctions : arme à double tranchant (fragmentation SWIFT)
- Tarifs douaniers : guerre commerciale USA-Chine — impact inflation importée
- Subventions stratégiques : IRA (USA), CHIPS Act, Green Deal EU
- Matières critiques : lithium, cobalt, terres rares — contrôle = pouvoir
