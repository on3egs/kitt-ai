# MODULE HISTOIRE DE L'INFORMATIQUE — Grandes Étapes

## Pionniers et fondations (1930–1960)
- **Alan Turing** (1912–1954) : père de l'informatique théorique. Machine de Turing (1936),
  test de Turing (1950). Décrypte Enigma (WWII). Condamné pour homosexualité, gracié en 2013.
- **John von Neumann** (1903–1957) : architecture von Neumann — CPU + mémoire + I/O — base
  de tous les ordinateurs modernes.
- **Grace Hopper** (1906–1992) : invente le premier compilateur (1952), co-crée COBOL.
  Amiral de l'US Navy. Première femme à recevoir le National Medal of Technology.
- **ENIAC** (1945) : premier ordinateur électronique général, 30 tonnes, 18 000 tubes.
- **Transistor** (1947) : Bell Labs (Shockley, Bardeen, Brattain) — remplace les tubes,
  révolutionne l'électronique.

## L'ère des mainframes (1960–1975)
- **IBM System/360** (1964) : premier système compatible à grande échelle — architecture
  qui influence tous les ordinateurs jusqu'à aujourd'hui.
- **Intel** (1968) : fondé par Gordon Moore et Robert Noyce. Moore's Law (1965) :
  doublement des transistors tous les ~2 ans.
- **Premier microprocesseur** : Intel 4004 (1971) — 2 300 transistors, 740 kHz.

## Révolution micro-informatique (1975–1990)
- **Altair 8800** (1975) : premier micro-ordinateur en kit grand public.
- **Apple I** (1976) puis **Apple II** (1977) : Steve Jobs + Steve Wozniak.
- **Microsoft** (1975) : Bill Gates + Paul Allen — BASIC pour Altair, puis MS-DOS.
- **IBM PC** (1981) : standardise l'architecture x86 (Intel 8088). Microsoft fournit MS-DOS.
- **Macintosh** (1984) : première interface graphique grand public (GUI). Publicité "1984".
- **GNU Project** (1983) : Richard Stallman lance le mouvement logiciel libre.

## Internet et le Web (1990–2000)
- **World Wide Web** (1991) : Tim Berners-Lee (CERN) invente HTTP + HTML. Web public.
- **Linux** (1991) : Linus Torvalds, noyau open source basé sur Unix. Révolution du libre.
- **Navigateurs** : Mosaic (1993), Netscape (1994), Internet Explorer (1995).
- **Google** (1998) : Larry Page + Sergey Brin — algorithme PageRank.

## Ère mobile et cloud (2000–2020)
- **iPod** (2001), **iPhone** (2007), **iPad** (2010) : Apple redéfinit l'électronique grand public.
- **Android** (2008) : Google / Linux mobile — domine le marché mondial (72%).
- **Cloud Computing** : Amazon AWS (2006), Google Cloud, Microsoft Azure.
- **Smartphones** dépassent les PC en ventes : 2011.

## Ère IA (2012–présent)
- **AlexNet** (2012) : deep learning + GPU NVIDIA — révolutionne la vision par ordinateur.
- **Transformer** (2017) : "Attention is all you need" (Google). Base de GPT, BERT, Claude.
- **ChatGPT** (2022) : OpenAI. 100 millions d'utilisateurs en 2 mois.
- **LLM sur edge** (2023–2025) : Llama, Qwen, Phi — IA locale sur Jetson, Raspberry Pi.
- **NVIDIA** : fournisseur dominant des GPU pour l'IA mondiale (H100, B200, Jetson Orin).
