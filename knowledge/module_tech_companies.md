# MODULE TECH COMPANIES — Microsoft, Apple, Linux, Google, Meta

## Microsoft
- **Fondé** : 1975 par **Bill Gates** (né 1955) et **Paul Allen** à Albuquerque (NM)
- **Produit fondateur** : BASIC pour Altair 8800, puis MS-DOS pour IBM PC (1981)
- **Windows** : Windows 1.0 (1985) → Windows 95 (révolution) → XP → 7 → 10 → 11
- **Office** : Word, Excel, PowerPoint — suite bureautique dominante mondiale
- **Azure** : cloud computing numéro 2 mondial (après AWS)
- **Rachat LinkedIn** (2016, 26 Md$), **GitHub** (2018, 7,5 Md$), **Activision** (2023, 69 Md$)
- **IA** : investissement majeur dans OpenAI (13 Md$), Copilot intégré partout
- **CEO actuel** : **Satya Nadella** (depuis 2014) — transformé Microsoft en leader cloud+IA

## Apple
- **Fondé** : 1976 par **Steve Jobs** (1955–2011), **Steve Wozniak** et Ronald Wayne
- **Produits emblématiques** : Mac (1984), iPod (2001), iPhone (2007), iPad (2010),
  AirPods (2016), Apple Watch, Vision Pro (2024)
- **Systèmes** : macOS (Unix-based), iOS, iPadOS, watchOS, tvOS
- **Puce Apple Silicon** (M1–M4, 2020+) : rupture x86, ARM custom ultra-performant
- **App Store** : 35 Md$ de revenus app en 2023
- **CEO actuel** : **Tim Cook** (depuis 2011 après mort de Jobs)
- **Capitalisation** : ~3 500 Md$ (2025) — régulièrement numéro 1 mondial

## Linux
- **Créé** : 1991 par **Linus Torvalds** (né 1969, Finlande), étudiant à Helsinki
- **Principe** : noyau open source (licence GPL) — tout le monde peut contribuer
- **Distributions principales** : Ubuntu, Debian, Fedora, Arch, CentOS/RHEL, Alpine
- **Domination serveur** : 96% des serveurs web mondiaux tournent sous Linux
- **Android** : basé sur le noyau Linux — 72% des smartphones
- **Supercomputers** : 100% des 500 supercalculateurs mondiaux
- **Jetson** : Ubuntu 22.04 / JetPack = Linux + drivers NVIDIA — base de KYRONEX
- **Linus Torvalds** : toujours mainteneur principal du noyau depuis 34 ans

## Google / Alphabet
- **Fondé** : 1998 par **Larry Page** et **Sergey Brin** (Stanford)
- **Moteur de recherche** : 92% de part de marché mondial
- **Android** (2008) : OS mobile dominant
- **YouTube** (racheté 2006, 1,65 Md$) : 2e moteur de recherche mondial
- **Google Cloud** : 3e cloud mondial
- **DeepMind** / **Google AI** : AlphaGo, Gemini, recherche IA de pointe
- **CEO actuel** : **Sundar Pichai** (depuis 2015)

## Meta (Facebook)
- **Fondé** : 2004 par **Mark Zuckerberg** à Harvard
- **Produits** : Facebook, Instagram (racheté 2012), WhatsApp (racheté 2014), Messenger
- **Metaverse** : pari massif (Quest VR, Horizon Worlds) — résultats mitigés
- **LLaMA** : modèles open source LLM (Llama 2, 3) — très utilisés en local (Jetson)
- **CEO** : Mark Zuckerberg (toujours fondateur-CEO)

## Rivalités historiques clés
- Gates vs Jobs (1980–2000) : DOS/Windows vs Mac — guerre des GUI
- Microsoft vs Netscape : guerre des navigateurs (1995–2001) — antitrust
- Apple vs Samsung : guerre des smartphones (2011+) — brevets design
- Google vs OpenAI/Microsoft : course à l'IA générative (2022+)
